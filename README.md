# 守望先锋弹钢琴

在守望先锋中自动弹奏钢琴——通过屏幕捕捉识别游戏内琴键，自动控制鼠标点击演奏。

# 下载链接
https://wwazp.lanzouw.com/iZcna3rb4yni

## 功能

- 屏幕捕捉识别守望先锋游戏窗口中的钢琴键盘
- 校准系统：标记参考键位，自动计算所有 65 键（C1-E6）的鼠标移动量
- 支持多种乐谱格式：JSON、MIDI、TXT
- 曲谱库管理（添加、删除、批量导入）
- 拟人化鼠标轨迹（曲线移动、随机抖动、可变点击时长）
- 全局热键（X 键）在游戏内即可开始/停止演奏
- FOV 投影模型：将像素偏移精确转换为鼠标移动计数

## 系统要求

- Windows 10 / 11（64 位）
- 守望先锋（窗口化或无边框窗口模式，16:9 分辨率）
- **管理员权限**（全局键盘钩子和鼠标模拟需要）
- Python 3.9+（仅源码运行需要）

## 乐谱格式

### JSON

```json
[
  {"note": "C4", "duration": 0.5},
  {"note": "D4", "duration": 0.5},
  {"note": "E4", "duration": 1.0}
]
```

### TXT

```
C4
D4
E4
```

每行一个音名，默认 0.5 秒时长。

### MIDI

支持 `.mid` / `.midi` 文件，自动解析所有轨道的 note_on 事件并计算时长。

### 音名规则

科学音高记法：`C4`、`D#4`、`Eb4`... 支持范围 C1 到 E6。

## 工作原理

守望先锋使用原始输入，忽略绝对鼠标定位。本程序通过发送**相对鼠标移动**来控制视角，使准星对准目标琴键并点击。

1. 截取游戏窗口屏幕截图
2. OpenCV 阈值分割 + 轮廓检测识别白色琴键区域
3. 用户校准标记参考键位，程序计算 FOV 投影参数
4. 播放时，将每个音符的屏幕坐标转换为鼠标移动增量
5. 通过 `SendInput` API 发送相对移动 + 点击

```
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│ 屏幕捕捉  │ →  │ 键位识别  │ →  │ FOV 投影  │ →  │ 鼠标控制  │
│  (mss)   │    │ (numpy)  │    │ (校准参数) │    │(SendInput)│
└──────────┘    └──────────┘    └──────────┘    └──────────┘
```

## 项目结构

```
守望先锋弹钢琴/
├── main.py           # PyQt5 GUI 主程序 + 三步操作流程
├── player.py         # 播放引擎（线程化，曲线移动）
├── mouse.py          # SendInput 鼠标控制（相对移动 + 点击）
├── calibrate.py      # 校准向导（WebView 截图标记）
├── projection.py     # FOV 投影模型（像素 → 角度 → 鼠标计数）
├── human_input.py    # 拟人化输入（曲线轨迹、随机抖动）
├── capture.py        # 屏幕捕捉（mss 窗口截图）
├── midi.py           # MIDI 文件解析 + 乐谱管理
├── build.spec        # PyInstaller 打包配置
├── build.bat         # 一键构建脚本
├── README.txt        # 用户使用说明（随 EXE 分发）
└── requirements.txt  # Python 依赖
```

## 注意事项

- 必须以**管理员身份**运行，否则键盘钩子和鼠标模拟无法对游戏生效
- 游戏必须设为**窗口化**或**无边框窗口**模式，全屏模式无法截取
- 改变游戏窗口大小或位置后需要**重新校准**
- 部分杀毒软件可能误报（键盘钩子 + 鼠标模拟），请添加信任
