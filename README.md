# 守望先锋弹钢琴 🎹

在守望先锋（Overwatch）游戏中自动演奏钢琴的 Windows 桌面工具。通过键盘驱动校准琴键位置，使用 `SendInput` API 控制鼠标演奏乐谱。

## 系统要求

## 操作流程

### 第一步：选择游戏窗口

程序启动后，左上角下拉框自动列出所有可见窗口。选择 Overwatch / 守望先锋 的窗口，点击「刷新」可重新扫描。

### 第二步：校准（键盘驱动）

这是最关键的一步——测量每个琴键相对于准星原点的鼠标位移量。

1. 点击「开始捕获」启动游戏画面实时预览
2. 点击「开始校准」进入键盘驱动校准模式
3. 使用**方向键**将准星移动到目标琴键中心：

   | 按键 | 步长 |
   |------|------|
   | `↑` `↓` `←` `→` | 粗调（10 鼠标单位） |
   | `Ctrl` + 方向键 | 微调（3 鼠标单位） |

4. 对准后按 **`H`** 键记录当前位置
5. 按照状态栏提示，依次标记全部 **65 个琴键**（C1 至 E6，38 白键 + 27 黑键）
6. 完成后校准数据自动保存到 `calibration.json`

**全局热键：**

| 按键 | 功能 |
|------|------|
| `X` | 开始 / 停止演奏 |
| `R` | 复位准星到校准原点 |

校准数据会自动保存，下次启动无需重新校准。如果改变游戏窗口大小或位置，需重新校准。

### 第三步：演奏

1. 在右侧面板点击「上传琴谱」，选择乐谱文件
2. 在列表中点击曲谱名称，右侧显示前 50 个音符预览
3. 点击「开始演奏」或切回游戏按 **`X`** 键开始
4. 再次按 `X` 停止

**播放控制：**

- **速度滑块**：0.25× ~ 10.00×
- **进度滑块**：拖动跳转到任意音符位置，松开后点击演奏即从该位置开始
- 演奏完成或手动停止后进度归零，可重新开始

## 乐谱格式

### JSON（`.json`）

```json
[
  {"note": "C4", "duration": 0.5},
  {"note": "D4", "duration": 0.5},
  {"note": "E4", "duration": 1.0}
]
```

### MIDI（`.mid` / `.midi`）

自动解析所有轨道，提取 `note_on` 事件并计算时值。

### TXT（`.txt`）

空白字符分隔的音名，每个音符默认 0.5 秒时长：

```
C4 D4 E4 F4 G4 A4 B4 C5
```

### 音名规则

使用科学音高记法（Scientific Pitch Notation）：

- **白键**：`C`, `D`, `E`, `F`, `G`, `A`, `B`
- **黑键**：`C#`, `D#`, `F#`, `G#`, `A#`
- **范围**：`C1` ~ `E6`（共 65 键）

## 技术原理

守望先锋使用 Raw Input，忽略绝对鼠标定位（`SetCursorPos` 等 API 无效）。本程序的解决方案：

1. **校准**：键盘驱动测量每个琴键相对于屏幕中心的像素位移 → 转换为鼠标计数
2. **演奏**：通过 `SendInput` API 发送**相对鼠标移动**（`MOUSEEVENTF_MOVE | MOUSEEVENTF_MOVE_NOCOALESCE`）
3. 游戏将位移解释为视角转动，准星随之移动到目标琴键
4. 每次点击后立即将准星移回原点，防止累积漂移

```
┌────────────┐    ┌──────────────────┐    ┌──────────────┐
│ 屏幕截图预览 │    │ 键盘驱动校准 (65键) │    │ SendInput    │
│   (mss)    │    │  方向键+H 标记     │    │ 相对移动+点击  │
└────────────┘    └──────────────────┘    └──────────────┘
```

## 常见问题

**Q: 点击播放后鼠标不动？**
A: 游戏显示模式必须为「窗口化」或「无边框窗口」，全屏模式无法截取画面。

**Q: 演奏时音符瞄准有偏差？**
A: 重新校准，确保每个标记点都对准琴键中心。

**Q: 按 X 键没有反应？**
A: 必须以管理员身份运行（exe 图标右下角应有盾牌标记）。

**Q: 杀毒软件报毒？**
A: 程序使用键盘钩子和鼠标模拟 API，可能被误报。请将程序文件夹添加为安全软件信任区。

**Q: 预览画面全黑？**
A: 确认选择了正确的游戏窗口，尝试将游戏设为无边框窗口模式。
